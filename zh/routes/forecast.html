Report ​

💡 Example: https://rsshub.app/outagereport/ubisoft/5

🛎️ Route: /outagereport/:name/:count?

🔗 Parameters:

  • name
    • Required
    • Description: Service name, spelling format must be consistent with URL
  • count
    • Optional
    • Description: Counting threshold, will only be written in RSS if the number of people who report to stop serving is not less than this number

🐙 Source Code: /outagereport/index.ts

Please skip the local service area code for name, for example https://outage.report/us/verizon-wireless to verizon-wireless.
